## Horoscope-Integrated Opening Prompt

**Category:** Class Opening/Theme  
**Status:** ✅ Validated (works very well)  
**Use Case:** Creating opening intros that weave multiple elements together  
**Validated By:** Tiff

### Prompt Text
```
Need an opening intro that weaves the description and affirmation by sharing a story that also matched the attached horoscope
```

### Context
- Combines narrative, affirmation, and seasonal/astrological context
- Creates cohesive class opening that feels both personal and timely
- Works especially well when horoscope or seasonal reference is provided
- Produces layered intros that anchor students in both personal experience and larger context

### How to Use
1. Provide horoscope or seasonal context reference
2. Share class description/theme
3. Include affirmation or core teaching point for the class
4. Use this prompt to generate opening that weaves all three together
5. Result: Rich, story-driven opening that sets the tone for the class

### Success Indicators
- Opening feels specific and authentic (not generic)
- Horoscope/seasonal element enhances without overwhelming
- Story creates bridge between abstract teaching and lived experience
- Affirmation lands with power and relevance

### Example Structure (Implied)
```
Story/Anecdote → [connects to] → Horoscope/Seasonal Context → [reinforces] → Affirmation/Teaching
```
